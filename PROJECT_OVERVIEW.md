## 🔌 API 设计

### 影视剧 API

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/series` | GET | 搜索影视剧 |
| `/api/series/:id` | GET | 获取详情 |
| `/api/series/:id/episodes` | GET | 剧集列表 |
| `/api/series/:id/cast` | GET | 演职人员 |
| `/api/series/:id/images` | GET | 图片列表 |
| `/api/series/trending` | GET | 热门影视剧 |

### 弹幕 API

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/episodes/:id/danmakus` | GET | 获取弹幕 |
| `/api/episodes/:id/danmakus` | POST | 发送弹幕 |
| `/api/episodes/:id/danmakus/merged` | GET | 合并弹幕 |
| `/api/danmaku/import` | POST | 导入弹幕 |
| `/api/danmaku/merge` | POST | 合并多平台弹幕 |
| `/api/episodes/:id/time-offset` | POST | 设置时间偏移 |
| `/ws/danmaku/:episode_id` | WS | 实时弹幕 |

## 🎨 前端设计

### 设计风格

- **深空黑主题** - 星空紫蓝主色调
- **玻璃态效果** - 毛玻璃 + 光影层次
- **流体动画** - Framer Motion + GSAP
- **电影质感** - 视差滚动 + 噪点纹理

### 核心页面

| 页面 | 功能 | 特色 |
|------|------|------|
| 首页 | 热门推荐、分类浏览 | Hero轮播、卡片悬浮光晕 |
| 搜索页 | 搜索、筛选 | 实时搜索、筛选动画 |
| 详情页 | 影视信息、剧集列表 | 视差背景、时间轴剧集 |
| 播放页 | 视频播放、弹幕 | 全屏沉浸、多源弹幕 |
| 用户中心 | 片单、弹幕记录 | 个人数据统计 |

### 动画效果

| 场景 | 动画 |
|------|------|
| 页面入场 | 幕布升起 - 模糊→清晰 |
| 卡片悬浮 | 光影浮现 - 浮起 + 光晕 |
| 弹幕流动 | 星河流淌 - 丝滑滚动 |
| 评分展现 | 星辰点亮 - 依次脉动 |
| 页面切换 | 胶片转场 |
